# Cline Rule: Axin Personal IP Illustrations

Read `axin-personal-ip-illustrations/SKILL.md` before changing or using this repository.

- Extract the main judgement or workflow first.
- Create a shot list for multi-image requests.
- Generate one concept per image.
- Keep the white hand-drawn style.
- Use 阿鑫 / Axin as the default and only recurring character.
