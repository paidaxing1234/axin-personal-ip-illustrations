# 授权海星方向角色说明

## 用途

这个仓库支持一个“授权海星方向”的个人 IP 角色分支，用于更轻松、更亲和、更适合外部传播的内容入口图。

用户已说明自己拥有相关授权，因此本仓库可以为该用户保留这个方向的角色提示词和示例资产。

## 对下游使用者的边界

如果你没有相关版权、商标或角色授权，不要直接复刻受保护角色。你可以：

- 改成普通原创海星。
- 去掉标志性服装和表情。
- 使用阿墨作为默认角色。
- 保留“热心、憨直、帮忙搬内容”的性格，但重新设计外形。

## 重做方向

- 更像一个可运营的个人 IP 主视觉，而不是流程图里的图标。
- 圆润、亲和、有表情，但不要幼稚。
- 保留“派大星方向”的授权识别度。
- 让它自然参与内容生产动作，例如抱着稿纸、指向多平台卡片、把项目压成资产。
- 少文字，少箭头，少机械结构。
