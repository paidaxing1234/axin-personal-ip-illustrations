# 授权海星方向主视觉 Prompt

用途：生成正式的授权海星方向个人 IP 主视觉。之前脚本手绘版本已被判定为不合格，不要复用。

## Prompt

```text
Use case: illustration-story
Asset type: public-facing hero illustration for a multi-platform personal IP illustration skill

Primary request:
Create a polished, friendly, expressive starfish-inspired personal IP character illustration for a Chinese AI content workflow brand. The user has rights to use a Patrick Star-like direction, so the character may clearly evoke a pink starfish with green shorts, but make it a bespoke personal IP mascot for this repository, not a direct frame copy.

Scene/backdrop:
Pure white clean background, no texture, no gradient, no shadows except extremely soft grounding if needed.

Subject:
A warm, rounded pink starfish character with green shorts, slightly goofy but charming expression, big friendly eyes, simple mouth, relaxed pose. The character is holding a small stack of real project notes in one hand and gently pointing toward three floating rounded cards labeled "Codex", "Hermes", and "GEO". The character should feel like a lovable operator helping turn messy experience into reusable public assets.

Style/medium:
High-quality modern 2D cartoon illustration with clean confident linework, soft flat color, polished but not corporate. Friendly personal IP character design, suitable for GitHub README hero, WeChat article header, and social profile material.

Composition/framing:
16:9 horizontal hero image. Character large on the left-center, floating platform cards on the right. Plenty of white space. The image should look like a clean product mascot hero, not a diagram. Keep all text short and legible.

Color palette:
Pink starfish body, green shorts, black clean linework, small orange connector accent, tiny red warning tag optional, blue accent only if needed.

Text:
Only these labels if text is used: "Codex", "Hermes", "GEO". No other text. No title.

Constraints:
Must look attractive and usable as a real personal IP hero asset. Avoid the previous ugly hard-edged mechanical look. Rounded, charming, professional, lively, polished. Do not create a dense workflow chart. Do not make it childish, cheap, scary, distorted, jagged, flat PowerPoint clipart, or rough programmer art. No watermark, no logo, no extra characters, no busy background, no long Chinese labels.

Negative:
ugly, awkward, angular star shape, jagged limbs, stiff pose, scary eyes, dead expression, cluttered diagram, too many arrows, low quality, screenshot, stock icon, PowerPoint, over-detailed background, text-heavy infographic, malformed hands, malformed eyes
```

## 验收

- 第一眼好看，能当主视觉。
- 明显是海星方向，亲和但不低幼。
- 有个人 IP 角色感，不是流程图图标。
- 有 Codex / Hermes / GEO 的多平台信息，但画面不拥挤。
- 不能像 `assets/rejected/` 里的草稿。
