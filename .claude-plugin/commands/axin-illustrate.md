# /axin-illustrate

Use the Axin personal IP illustration workflow.

## Instructions

1. Read `skills/axin-personal-ip-illustrations/SKILL.md`.
2. Read the user-provided article or topic.
3. Produce a concise shot list.
4. Generate one image per selected concept.
5. Save final assets under `assets/<topic-slug>-illustrations/`.

Use 阿鑫 / Axin as the default and only recurring character. Keep the image clean, strange, white-background, and focused on one core action.
