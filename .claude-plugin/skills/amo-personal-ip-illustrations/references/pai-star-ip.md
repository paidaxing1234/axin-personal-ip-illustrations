# 授权海星方向角色

## 角色定义

这是为已授权的海星方向个人 IP 准备的 public-facing 角色分支。它适合用于更轻松、更有亲和力的内容封面、公众号配图、视频封面和多平台入口图。

如果使用者没有相关授权，应将它替换为普通原创海星角色，不要复刻特定受保护角色的名称、服装、表情和标志性细节。

## 外形方向

- 粉色海星轮廓，五角身体，姿态松弛。
- 表情憨直、热心、稍微发呆，但不要低幼化。
- 可以穿绿色短裤方向的轻松装束；如果要避开授权风险，可改成工作围裙、腰包或橙色线缆腰带。
- 和阿墨共享“内容工坊”世界观：可以搬稿纸、接线、贴标签、推发布箱。
- 线条仍然保持手绘、留白、少色，不要变成复杂卡通海报。

## 性格

- 外向、热心、行动比理解快半拍。
- 适合承担“把复杂内容讲得轻一点”的动作。
- 可以作为读者入口角色，阿墨则更像后台质检角色。

## 常见职责

- 把散乱想法抱进内容工坊。
- 举着“先跑起来”的标牌，但旁边必须有证据或输出。
- 把多平台入口连成一串：Codex、Hermes、Claude、Generic Agents。
- 把一个项目拆成 README、短文、视频脚本和配图资产。
- 在发布前把过度包装的内容拦下来，换成真实结果。

## 禁止

- 不要直接声称该角色可供所有人自由使用受保护 IP。
- 不要在无授权场景复刻受保护角色的完整造型。
- 不要只画搞笑表情或贴纸，必须服务内容结构。
- 不要让画面变成儿童卡通、商业海报或表情包。

## 判断标准

读者应该先感到亲近，然后 1 秒内看懂它正在推动哪一个个人 IP 流程。如果只记住了角色，没有记住结构，说明图偏了。
