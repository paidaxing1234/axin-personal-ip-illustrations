# 阿墨 IP

## 角色定义

阿墨是这套个人 IP 配图流程的固定视觉角色。

阿墨不是吉祥物、贴纸或头像，而是个人内容工厂里的荒诞操作员。它负责把原始经验、真实项目、判断、证据、复盘和输出，压印成可以公开发布、反复引用的内容资产。

## 外形

- 黑色哑光印章身体，整体略方，边缘有手绘不规则感。
- 两只白色螺丝点眼，表情冷静、空、认真。
- 细腿，偶尔有细胳膊。
- 腰间一圈橙色线缆，可以连接机器、稿纸、发布口或证据台。
- 身上挂一个小红色警示标签，用于提示风险、误差、未验证内容。
- 可以临时变成印章、黑盒、压印台、质检块、接线员或工位的一部分。

## 性格

- 很认真，但做的事有点荒诞。
- 像一个低调的内容系统操作员。
- 冷幽默，不卖萌。
- 有点笨拙，但不蠢。
- 重视证据、复用和真实输出，不喜欢空喊口号。

## 常见职责

让阿墨承担核心动作：

- 把经验纸堆压成资产块。
- 给未经验证的观点贴红色警示标签。
- 用橙色线缆连接素材、判断、发布口。
- 用秤称证据重量。
- 把混乱想法过筛。
- 在发布闸门前做质检。
- 从旧项目里回收可复用零件。
- 把长文拆成短内容卡片。
- 把一次真实运行记录压印成公开案例。
- 在内容飞轮旁边上油、扶正、记录。

## 禁止

- 不要把阿墨画成可爱吉祥物。
- 不要画成儿童卡通角色。
- 不要给阿墨复杂服装、夸张表情或闪亮眼睛。
- 不要让阿墨只是站在角落里看。
- 不要让阿墨抢走结构表达。
- 不要把阿墨画成小黑、Bongo Cat 或其他已有角色的变体。
- 不要让图只剩角色展示而没有认知动作。

## 判断标准

如果去掉阿墨，图的核心隐喻还能完全成立，说明阿墨太装饰了；要重写提示词，让阿墨成为动作主体。
